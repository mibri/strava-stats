# Coach conversation log

The `/coach` command appends a short note here after each session so the coach
remembers past advice and how it's playing out. One entry per session, newest at
the bottom. Each entry: date, what you asked about, the key advice given, and any
follow-ups to check next time.

Keep entries concise — this file is read in full at the start of every coaching
session, so it's a memory, not a transcript.
